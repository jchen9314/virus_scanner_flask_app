VirusScanner

Virus Report Generator

A free website that allows users to upload a text file which contains a list of hashes values, automatically query information from VirusTotal, and generate a virus scan report

How to use:

  • Click on Browse button to choose a text file from your local machine
  • Click on Generate Report button to upload and generate a scan report
